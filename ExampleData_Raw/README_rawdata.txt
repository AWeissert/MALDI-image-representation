README - Rawdata (created 20 July 2026)

One Folder for each Tumor.
Each Folder contains three items:
- [...]colData.csv: contains MALDI intensities for each spot
- [...]Reg.csv: contains x-y-coordinates for each spot
- [...].jpg: the HER2-coloured microscope Image
